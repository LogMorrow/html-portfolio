# Lagos Property Nexus

A full-stack real estate property listing marketplace for Lagos, Nigeria.

## Features
- Search and browse properties for sale or rent.
- Property verification and diaspora-ready badges.
- Investment intelligence dashboard with market metrics.
- Submit property listings for verification.
- Submit buyer/renter inquiries.
- SEO-ready landing page metadata and performance-conscious UI.
- API hardening with secure headers and basic rate limiting.

## Run locally
```bash
npm start
```

Then open `http://localhost:3000`.
